Python Коментарі

❮ Prev Next ❯

Коментарі можуть бути використані для пояснення коду Python.

Коментарі можна використовувати, щоб зробити код більш читабельним.

Коментарі можна використовувати для запобігання виконанню під час тестування коду.

Створення коментаря

Коментарі починаються з символа "решітка" #, і до кінця рядка Python буде їх ігнорувати:

Приклад

#Це коментар
print("Hello, World!")
Спробуйте самі »

Коментарі можна розміщувати в кінці рядка, а Python ігноруватиме решту рядка:

Приклад

print("Hello, World!") #Це коментар
Спробуйте самі »

Коментарі не обов'язково повинні бути лише для пояснення коду, вони також можуть використовуватися для запобігання Python виконувати код:

Приклад

#print("Hello, World!")
print("Cheers, Mate!")
Спробуйте самі »

Багаторядкові коментарі

Python насправді не має синтаксису для багаторядкових коментарів.

Щоб додати багаторядковий коментар, ви можете вставити # для кожного рядка:

Приклад

#Це коментар
#написаний на
#більше ніж одному рядку
print("Hello, World!")
Спробуйте самі »

Або, не зовсім за призначенням, ви можете використовувати багаторядковий рядок.

Оскільки Python буде ігнорувати рядкові літерали, не присвоєні змінній, ви можете додати багаторядковий рядок (потрійні лапки) у свій код і помістити в нього свій коментар:

Приклад

"""
Це коментар
написаний на
більше ніж одному рядку
"""
print("Hello, World!")
Спробуйте самі »

Поки рядок не призначений змінній, Python буде читати код, але потім ігноруватиме його, якщо ви зробили багаторядковий коментар.
