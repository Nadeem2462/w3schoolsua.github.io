Завантажити Node.js

На офіційному вебсайті Node.js є інструкції зі встановлення Node.js: https://nodejs.org

Початок роботи

Після того, як ви завантажили та встановили Node.js на своєму комп’ютері, давайте спробуємо відобразити "Hello World" у веб-браузері.

Створіть файл Node.js під назвою "myfirst.js" і додайте наступний код:

myfirst.js

var http = require('http');

http.createServer(function (req, res) {
  res.writeHead(200, {'Content-Type': 'text/html'});
  res.end('Hello World!');
}).listen(8080);

Збережіть файл на своєму комп’ютері: C:\Users\Your Name\myfirst.js

Код повідомляє комп’ютеру написати "Hello World!" якщо хтось (наприклад, веб-браузер) намагається отримати доступ до вашого комп’ютера через порт 8080.

Поки що вам не потрібно розуміти код. Це буде пояснено пізніше.

Інтерфейс командного рядка

Файли Node.js потрібно ініціювати в "інтерфейсі командного рядка" програми вашого комп’ютера.

Як відкрити інтерфейс командного рядка на вашому комп’ютері, залежить від операційної системи. Для користувачів Windows натисніть кнопку "Пуск" і знайдіть "Командний рядок" або просто напишіть "cmd" у полі пошуку.

Перейдіть до теки, яка містить файл "myfirst.js", вікно інтерфейсу командного рядка має виглядати приблизно так:

C:\Users\Your Name>_

Запустіть файл Node.js

Файл, який ви щойно створили, має бути ініційований Node.js перед виконанням будь-яких дій.

Запустіть інтерфейс командного рядка, напишіть node myfirst.js і натисніть Enter:

Ініціювати "myfirst.js":

C:\Users\Your Name>node myfirst.js

Тепер ваш комп’ютер працює як сервер!

Якщо хтось спробує отримати доступ до вашого комп’ютера через порт 8080, він отримає повідомлення "Hello World!" повідомлення у відповідь!

Запустіть веб-браузер і введіть адресу: http://localhost:8080
